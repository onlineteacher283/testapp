# 📱 LearnNova Mobile App

LearnNova-এর অফিসিয়াল অ্যান্ড্রয়েড অ্যাপ্লিকেশনে আপনাকে স্বাগতম। এই অ্যাপটি শিক্ষার্থীদের অনলাইন পরীক্ষা এবং শিক্ষামূলক কন্টেন্ট সহজে অ্যাক্সেস করার জন্য তৈরি করা হয়েছে।

---

## 🚀 বৈশিষ্ট্যসমূহ (Features)

* **সহজ নেভিগেশন:** ব্যবহারকারী বান্ধব ইন্টারফেস।
* **অনলাইন পরীক্ষা:** সরাসরি অ্যাপের মাধ্যমে কুইজ এবং পরীক্ষায় অংশগ্রহণ।
* **সিকিউর লগইন:** OTP ভেরিফিকেশন সিস্টেম।
* **ডাইরেক্ট সাপোর্ট:** সরাসরি আমাদের ইনস্টিটিউটের সাথে যোগাযোগের সুবিধা।
